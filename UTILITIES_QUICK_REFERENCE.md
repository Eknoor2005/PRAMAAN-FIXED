# Utilities Tool - Quick Reference

## Navigation
**Path:** Dashboard → Utilities & Tools (Sidebar)
**URL:** `/dashboard/utilities`

---

## Three Main Tools

### 1️⃣ Camera Recorder
**Tab:** Camera | **Icon:** 📹

#### Quick Steps:
1. Click "Start Camera"
2. Allow camera permissions
3. Click "Start Recording"
4. Speak your testimony
5. Click "Stop Recording"
6. Click "Download Video"

#### File Output:
- Format: WebM video
- Quality: 1280x720
- Audio: Synchronized
- Codec: VP8/Opus

#### Tips:
✅ Good lighting
✅ Quiet location
✅ Face camera
✅ Speak clearly

---

### 2️⃣ Microphone Recorder
**Tab:** Microphone | **Icon:** 🎙️

#### Quick Steps:
1. Click "Start Recording"
2. Allow microphone permissions
3. Speak your testimony
4. Watch the timer
5. Click "Stop Recording"
6. Play back to verify
7. Click "Download"

#### File Output:
- Format: WebM audio
- Quality: 128kbps
- Sample Rate: 48kHz
- Codec: Opus

#### Tips:
✅ Silent environment
✅ Consistent volume
✅ Pause between thoughts
✅ Test first

---

### 3️⃣ Translator
**Tab:** Translator | **Icon:** 🌍

#### Quick Steps:
1. Enter text in left box
2. Select source language (auto-detect)
3. Choose target language
4. Click "Translate"
5. View translation on right
6. Click "Copy Translation"

## Common Issues & Solutions

### Camera Won't Start
```
❌ Problem: Camera blocked
✅ Solution: Check browser permissions
  1. Click lock icon in address bar
  2. Allow camera access
  3. Refresh page
```

### No Audio Recording
```
❌ Problem: Microphone off
✅ Solution: Enable microphone
  1. Check system settings
  2. Verify microphone connected
  3. Test in browser
```

### Translation Slow
```
❌ Problem: Network lag
✅ Solution: Check connection
  1. Test internet speed
  2. Reduce text length
  3. Try again
```

### File Sizes
- 5 min video: ~15-20 MB
- 5 min audio: ~3-4 MB

## FAQ

**Q: Is my recording saved?**
A: No, you must download or upload to vault.

**Q: Can I edit my recording?**
A: No, but you can re-record.

**Q: What if I stop before finishing?**
A: Recording will be saved. Start new one.

**Q: Can I translate back to original language?**
A: Yes, use "Swap" button.

**Q: How long can recordings be?**
A: No hard limit, but recommend <30 min.

**Q: Is translator free?**
A: Yes, powered by Google Translate.

**Q: Can I download translations?**
A: Copy text and paste into document.

**Q: What if translation is wrong?**
A: Review, correct, or try different language.
